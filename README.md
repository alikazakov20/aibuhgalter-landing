# AI-Бухгалтер — Landing Page

Заглушка для домена [aibuhgalter.com](https://aibuhgalter.com) — нейропомощник по налогам для ИП на УСН.

## Что это

Single-page лендинг до запуска основного веб-приложения (через 4 недели).

**Функции:**
- Информация о продукте и тарифах (290 / 990 / 1990 ₽/мес)
- Сравнение с конкурентами (Эльба, Моё Дело)
- FAQ с ответами на основные возражения
- CTA на Telegram-бот: [@AI_Buxgalter_USN_bot](https://t.me/AI_Buxgalter_USN_bot)

## Стек

- Чистый HTML + Tailwind CSS через CDN
- Шрифт Montserrat (Google Fonts)
- Без сборки, без Node.js — деплой одним файлом
- Хостинг: [Vercel](https://vercel.com)

## Локальный запуск

Просто открой `index.html` в браузере. Всё.

```bash
open index.html
```

## Деплой

Коммит в `main` → Vercel автоматически обновляет сайт.

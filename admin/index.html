Смена PIN-кода

                
PIN хранится в env vars Vercel. Смена пока через CLI.

                
vercel env rm ADMIN_PIN_HASH production
echo -n "НОВЫЙ_PIN" | shasum -a 256 | awk '{print $1}'
# вставь хеш
printf 'ВСТАВЬ_ХЕШ\n' | vercel env add ADMIN_PIN_HASH production
vercel deploy --prod --yes

                
Скоро: смена через UI этой страницы (требует доп. endpoint).
